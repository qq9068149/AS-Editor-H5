# AS-Editor-H5

#### 介绍
基于 vue 可视化 [AS-Editor](https://gitee.com/was666/as-editor.git) 的H5展示模板，与 [AS-Editor](https://gitee.com/was666/as-editor.git) 一起使用可以可视化生成vue页面

#### 安装教程

1.  npm install （安装 node_modules 模块）
2.  npm run serve （运行）
3.  npm run build （打包）

#### 使用说明
> home.vue 页面为demo实例

> 已配置好vant ui 按需引入
页面中需要使用的话 import { 组件 } from 'vant';
