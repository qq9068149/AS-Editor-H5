# AS-Editor-H5

#### Introduce

Vue-based mobile template that can generate pages from defined JSON.

#### 搭配

Using with [AS-Editor](https://github.com/qq9068149/AS-Editor) makes it easy to generate VUE pages

The AS-Editor links：[https://github.com/qq9068149/AS-Editor](https://github.com/qq9068149/AS-Editor)

#### Install

1.  npm install （Install node_modules）
2.  npm run serve （run）
3.  npm run build （build）
